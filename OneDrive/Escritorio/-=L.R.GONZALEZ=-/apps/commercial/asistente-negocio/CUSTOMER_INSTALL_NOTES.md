# Notas De Instalacion Para Clientes

Producto: Asistente de Negocio MEDIOEVO v1.0.0

## Windows

- Usa el instalador `Asistente-Negocio-MEDIOEVO-1.0.0-win-x64.exe`.
- El instalador es de usuario actual y no debe pedir permisos de administrador.
- La app se instala en `%LOCALAPPDATA%\Programs\asistente-negocio-medioevo`.
- Tambien hay un ZIP portable para revision manual.

## Aviso De Firma

La build local validada no esta firmada digitalmente todavia. Windows puede
mostrar SmartScreen o una advertencia de editor desconocido.

Decision pendiente antes de checkout publico:

- firmar el instalador con certificado; o
- venderlo solo como founder access con aviso claro de instalador sin firma.

## Datos

- La app guarda los datos en el equipo local del usuario.
- El envio final ocurre en correo o WhatsApp del usuario.
- La app no envia mensajes sin aprobacion humana.
- El usuario puede descargar respaldo o borrar datos desde la app.
