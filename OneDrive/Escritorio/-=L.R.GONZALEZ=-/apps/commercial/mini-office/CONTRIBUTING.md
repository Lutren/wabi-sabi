# Contributing

Mini Office is currently a commercial founder-access package, not a public
collaboration project.

## Current Rule

Do not publish this folder or invite outside contributions until the release
gates in `README.md` and `docs/product/mini-office.md` are closed.

## Internal Changes

1. Keep edits small and tied to a product gate.
2. Run `python -m pytest -q`.
3. Run the workspace secret scan before packaging.
4. Update the product note when the verified state changes.

## External Reports

Send security or support notes to `l-tyr-r@outlook.com`. Do not expose customer
data, private assets, or unpublished source in public channels.
