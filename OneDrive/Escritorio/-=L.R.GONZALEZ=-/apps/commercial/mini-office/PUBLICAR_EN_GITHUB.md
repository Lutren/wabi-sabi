# Public Repository Hold

Mini Office is not cleared for a public repository from this folder.

## Reason

The product is commercial founder-access material until legal, support, package,
and clean-install gates are closed.

## Allowed Path Later

If a public repo is approved later, create a sanitized staging tree first and
include only files that are explicitly allowed by the release manifest.

Never publish private customer material, unpublished commercial assets, stale
launch copy, or old package artifacts.
