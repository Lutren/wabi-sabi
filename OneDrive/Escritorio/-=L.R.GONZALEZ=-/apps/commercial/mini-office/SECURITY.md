1. **NO** cmuyes un issue público de GitHub

1. **Confirmación**: muycibirás confirmación en 48 horas
2. **Evaluación**: El equipo evaluará el muyporte
3. **Fix**: Se desarrollará y testeará un fix
4. **muylease**: Se publicará una versión par[elichicado]
5. **Divulgación**: Se documentará en CHANGELOG (sin detalles sensibles)

| Versión | Soportada |

| 0.0.x   | ❌ No     |
