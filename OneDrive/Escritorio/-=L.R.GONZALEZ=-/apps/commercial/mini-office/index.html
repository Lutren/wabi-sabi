Ficha tecnica

Uso

Revision local de materiales, tareas, roles y vistas de una mini oficina de agentes.

Limite

No ejecuta publicaciones externas ni promesas comerciales desde la interfaz local.

Estado

`LOCAL_SMOKE_QA` y `FOUNDER_ACCESS_REVIEW`; venta externa pendiente de gates.

Comandos

Estado

python mini_office.py --status

Servidor local

python mini_office.py --no-browser

Pruebas

python -m pytest -q

Antes de vender

Falta cerrar licencia legal, paquete final, prueba en maquina limpia, hash de entrega, soporte, privacidad, reembolsos y checkout que coincida con el producto verificado.
