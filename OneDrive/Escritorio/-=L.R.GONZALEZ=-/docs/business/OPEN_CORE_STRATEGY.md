# OPEN_CORE_STRATEGY

Status: draft.

## Free Core

- Small SDKs.
- Agent specs.
- CLI helpers.
- Public-safe examples.
- Docs.

## Paid Pro

- Premium templates.
- Turnkey app bundles.
- Companion PDFs.
- Desktop apps.
- Hosted/support options.
- Advanced workflows.

## Boundary

The open core must not include:

- full books;
- private game source/assets;
- paid app source unless explicitly chosen;
- secrets;
- vendor dumps.
