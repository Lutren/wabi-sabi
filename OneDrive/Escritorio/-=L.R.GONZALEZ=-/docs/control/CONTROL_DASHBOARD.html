Opciones del sistema

Host APPROVE

Activar/verificar VPN WARP

Mantiene la red en Cloudflare WARP y verifica estado.

warp-cli connect; warp-cli status
Host APPROVE

Verificar geolocalizacion publica

Confirma warp=on, colo y pais sin tocar archivos.

Invoke-RestMethod https://www.cloudflare.com/cdn-cgi/trace
Host REVIEW

Guard geo/privacidad Claudio

Compara WARP, IP publica, Windows location y permisos de navegador.

python tools\security_geolocation_guard.py --connect --reported-google-location "Yucatan" --write --json
Curador APPROVE

Refrescar pendientes

Actualiza snapshot de backlog; no cierra tareas.

python tools\release\pending_review.py --write --quiet
COMMS APPROVE

Validar protocolo de agentes

python COMMS\tools\validate_seto_comms.py --json --fail-on-errors
