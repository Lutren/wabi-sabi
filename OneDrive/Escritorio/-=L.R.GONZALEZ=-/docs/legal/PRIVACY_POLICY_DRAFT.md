## Data Collected

Depends on product and platform. Possible data:

- purchase email through Gumroad or payment provider;
- support messages;
- app telemetry only if explicitly enabled;
- crash logs only if explicitly collected.

## Data Not Intended

- private game source access;
- unnecessary biometric data;
- secrets or API keys;
- user files unless the app explicitly processes local files.

## Required Before Launch

- product-specific data flow review;
- analytics decision;
- payment provider terms review;
- retention policy.
