# Publication Portfolio Map - MEDIOEVO / Claudio / Observacionismo

Fecha: 2026-05-05

Estado: canon publico operativo para separar que se muestra, que se vende y que queda privado.

## Regla central

MEDIOEVO publica herramientas, fichas, demos sinteticas, whitepapers de bajo claim y rutas comerciales claras. No publica la ingenieria privada completa, prompts internos, libros completos sin aprobacion, RPG/TCG, datasets privados, secretos, rutas locales ni automatizacion sensible.

## Bloqueos permanentes para publicacion general

- RPG/TCG, lore no aprobado, assets privados, game bridge y builds internos.
- Libros completos, vaults, prompts internos, datasets, rutas locales, historiales, sesiones, credenciales o tokens.
- Claims de diagnostico medico, prediccion social real, seguridad garantizada, nueva fisica probada o conciencia validada.
- Herramientas ofensivas, pentest repos, vendors no licenciados o automatizacion externa sin gate.

## Validacion antes de cualquier cambio live

1. Secret scan focalizado sobre archivos o listing copy.
2. Path scrub sin rutas privadas.
3. Claim scan sin promesas fuertes.
4. Link check del destino publico.
5. ActionGate por destino.
6. Verificacion post-publicacion antes de declarar el cambio como live.
