# game-private

Private boundary for the MetaEvo/TCG/videojuego layer.

No active game source was moved here in this pass. The active game remains under
its current owner/worktree until a coordinated private migration happens.

Rules:

- Do not copy `metaevo-tcg`, `claudio\tcg`, `runtime\game_bridge`, game builds,
  lore or private assets into public packages.
- Do not include this layer in `releases/free-dev`.
- Treat all game material as proprietary and private unless an explicit release
  decision says otherwise.
