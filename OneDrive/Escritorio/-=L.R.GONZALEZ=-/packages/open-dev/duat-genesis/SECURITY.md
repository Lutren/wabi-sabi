# Security

DUAT Genesis is local-only and dependency-free.

## Boundaries

- No network access.
- No filesystem writes except caller-directed output redirection.
- No shell execution.
- No browser, mouse or keyboard control.
- No loading of private data by default.

## Reporting Issues

Report issues with a minimal synthetic reproduction. Do not attach secrets,
private MEDIOEVO material, RPG/TCG files, customer data or medical data.
