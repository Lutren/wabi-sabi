# Private Exclusions

This package is intentionally open-dev and public-safe. It must not include:

- MEDIOEVO books, full canon, unpublished essays or editorial vaults.
- RPG, TCG, game bridge, lore, scenes, assets, balance data or private runtime code.
- Raw source notes, personal context, account inventory or family/private planning.
- Real credentials, tokens, cookies, browser profiles, `.env` files, cloud configs or session state.
- Vendor dumps, offensive-security repos, local model weights, build outputs or caches.
- Gumroad, Stripe, Discord, YouTube, GitHub account automation or deployment secrets.

Allowed content is limited to source code, tests, synthetic examples, docs, package metadata and local-only demonstration flows.
