# DUAT Templates

Paid template pack for teams that want to use DUAT Genesis as a public-safe
synthetic lab without copying MEDIOEVO private engineering.

## Includes

- DUAT Genesis lab card template.
- Synthetic scenario brief.
- Falsifier report template.
- Example synthetic lattice study.
- Gumroad listing draft.
- Commercial-use and support notes.

## Excludes

- DUAT Geodia private runtime, source, fixtures or bridge code.
- MEDIOEVO RPG/TCG assets, lore, scenes or game runtime.
- Real medical, biological, neurological, market or social datasets.
- Customer data, local sessions, prompts, credentials or API keys.
- Scientific, medical, safety or revenue guarantees.

## Use

Use these templates to plan synthetic runs, record assumptions, document
falsifier results and decide what still needs real validation before public
claims.
