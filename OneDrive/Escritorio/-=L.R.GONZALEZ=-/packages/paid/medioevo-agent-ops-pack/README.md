# MEDIOEVO Agent Ops Pack

Premium templates and operating checklists for building agent products around
open MEDIOEVO/Observacionismo tooling without exposing private runtime,
books, RPG/TCG material, credentials or local workspace data.

## Includes

- Technical card template for agent products.
- Curator report template with CERTEZA / INFERENCIA / INCOGNITA sections.
- ActionGate checklist for external actions.
- GitHub public release checklist.
- Gumroad listing template.
- Synthetic example agent audit.
- Support and commercial-use notes.

## Excludes

- Claudio private runtime.
- MEDIOEVO books, canon vault, RPG, TCG, assets and lore.
- Customer data, family data, local sessions and credentials.
- Browser, shell, Gumroad or GitHub automation credentials.
- High-risk scientific, medical, cognitive, legal, security or revenue promises.
